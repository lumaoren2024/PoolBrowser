## ===教程开始===
按照自己的操作系统修改对应的配置``config.json``

``chrome_path`` 是指纹池浏览器的用户存储数据

``chrome_app`` 是你的chrome浏览器的安装路径

``extensions`` 是指纹池浏览器的预加载插件路径

``user_agents`` 是指纹池浏览器的伪装用户头

``langs`` 是指纹池浏览器的伪装语言

``tzs`` 是指纹池浏览器的伪装用户时区
